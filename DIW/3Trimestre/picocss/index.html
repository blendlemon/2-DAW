PICO CSS✨

Pico CSS es un framework CSS minimalista y ligero diseñado espacificamente para maquetar sitios web utilizando HTML semántico sin necesidad de sturar el código con clases. Se enfoca en la simplicidad, ofreciendo estilos predefinidos y responosivos de forma autómatica con solo incluir la biblioteca
Está principlamente pensado pra utilizarse en webs html estáticas, landing pages, webs minimalistas y proyectos web pequeños
